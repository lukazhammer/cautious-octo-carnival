### 5. TECHNICAL SEO SPECIFICATIONS

#### URL Structure

**Format:** `/blog/[pillar-slug]/[post-slug]/`

**Examples:**
- `/blog/fundamentals/why-70-percent-small-businesses-fail/`
- `/blog/strategy-framework/brand-positioning/`
- `/blog/how-to-guides/define-brand-vision/`
- `/blog/comparisons/diy-vs-agencies/`
- `/blog/top-lists/top-10-branding-mistakes/`

**Implementation rules:**
- All lowercase
- Hyphens (not underscores)
- No dates in URL
- Keep under 75 characters
- Include primary keyword when possible

#### Canonical Tags

Every post includes self-referencing canonical:
```html
<link rel="canonical" href="https://brandora.com/blog/[pillar]/[slug]/" />
```

**Purpose:** Prevent duplicate content issues, consolidate link equity

#### XML Sitemap

Generate `/sitemap-blog.xml` with:

```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://brandora.com/blog/fundamentals/why-70-percent-small-businesses-fail/</loc>
    <lastmod>2025-11-15</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.8</priority>
  </url>
  <!-- Repeat for all 60 posts -->
  <!-- Hub posts: priority 0.9 -->
  <!-- Supporting posts: priority 0.7-0.8 -->
</urlset>
```

**Update frequency:** Weekly (auto-regenerate on publish/update)
**Submit to:** Google Search Console, Bing Webmaster Tools

#### Image Optimization Guidelines

**Filename convention:**
`[post-slug]-[descriptor]-[number].jpg`

Examples:
- `brand-positioning-perceptual-map-1.jpg`
- `five-step-process-diagram-1.jpg`
- `top-branding-mistakes-infographic-1.jpg`

**Alt text format:**
"[Descriptive text 8-12 words including keyword when relevant]"

Examples:
- "Brand positioning perceptual map showing competitive differentiation axes"
- "Five-step brand strategy process diagram from discovery to guidelines"
- "Infographic listing top 10 branding mistakes small businesses make"

**Technical specs:**
- Format: WebP with JPG fallback
- Size: < 200KB per image
- Dimensions: 1200px wide max (responsive)
- Compression: 80-85% quality

#### Mobile Optimization Checklist

- [ ] All posts render properly on mobile (320px-768px width)
- [ ] Font sizes: minimum 16px body text
- [ ] Touch targets: minimum 44x44px
- [ ] Images responsive (max-width: 100%)
- [ ] No horizontal scrolling
- [ ] Fast load time (< 3 seconds)
- [ ] Readable line length (50-75 characters)

#### Page Speed Targets

**Core Web Vitals:**
- **LCP (Largest Contentful Paint):** < 2.5s
- **FID (First Input Delay):** < 100ms
- **CLS (Cumulative Layout Shift):** < 0.1

**Optimization tactics:**
- Lazy load images below fold
- Minify CSS/JS
- Use CDN for assets
- Enable browser caching
- Compress text files (gzip/brotli)
