## Template Usage Guide

### How to Customize Templates

**Social Media Templates:**

1. Replace [bracketed placeholders] with actual content
2. Adjust length based on platform (Twitter 280 char, LinkedIn 1,300 char sweet spot)
3. Test different hooks‚Äîtrack which get highest engagement
4. Create platform-specific versions (don't just cross-post)

**Email Templates:**

1. Personalize with subscriber data ([First Name], [Industry], etc.)
2. A/B test subject lines
3. Track which CTAs get highest click-through
4. Segment by interest when possible

**Outreach Templates:**

1. Research each target before sending
2. Customize opening paragraph
3. Reference specific content from their site
4. Keep emails under 150 words
5. Always provide value before asking for anything

### Template Testing Strategy

**Week 1-2: Baseline**
- Use templates as written
- Track engagement metrics
- Note which perform best

**Week 3-4: Optimization**
- Test variations of top performers
- Adjust hooks, CTAs, formatting
- Track performance vs. baseline

**Month 2+: Personalization**
- Develop platform-specific variations
- Create audience segment variations
- Refine based on data

- Engagement rate: >2% (Twitter), >5% (LinkedIn)
