# Visual Identity System

## Logo Specifications

### Primary Logo

The Brandora primary logo consists of a custom wordmark with an integrated brand mark. The logo represents clarity, strategy, and transformation.

#### Primary Logo - Full Color
**Usage:** Primary usage on white or light backgrounds

```
┌─────────────────────────────────────┐
│                                     │
│    ◇ BRANDORA                      │
│                                     │
└─────────────────────────────────────┘
```

**Color specifications:**
- Wordmark: Navy #070058
- Brand mark (diamond): Blue #6F8AEC
- Background: White or #F1F9FF

**File formats:**
- Vector: `brandora-logo-primary.svg`
- Print: `brandora-logo-primary.pdf`, `brandora-logo-primary.eps`
- Digital: `brandora-logo-primary@1x.png`, `brandora-logo-primary@2x.png`, `brandora-logo-primary@3x.png`

---

### Icon/Symbol Mark

The Brandora diamond symbol can be used independently when brand recognition is established.

```
    ◇
  BRANDORA
```

**Symbol meaning:**
- Diamond shape: Value, precision, clarity
- Upward orientation: Growth, aspiration
- Geometric form: Strategic thinking, structure

**Usage:**
- App icons
- Social media profile images
- Favicons
- Small-scale applications
- Watermarks

**Minimum size:**
- Digital: 32×32 pixels
- Print: 0.5 inches (12.7mm)

---

### Logo Clear Space

Maintain clear space around the logo equal to the height of the letter "A" in Brandora.

```
        ┌─ A ─┐
        ↓     ↓
    A → ◇ BRANDORA ← A
        ↑     ↑
        └─────┘
          A
```

**Minimum clear space:**
- Top: 1A
- Right: 1A
- Bottom: 1A
- Left: 1A

---

### Minimum Size Requirements

#### Digital Applications
- **Primary logo (horizontal):** 150px width minimum
- **Icon only:** 32px × 32px minimum
- **Favicon:** 16px × 16px, 32px × 32px, 64px × 64px

#### Print Applications
- **Primary logo:** 1.5 inches (38mm) width minimum
- **Icon only:** 0.5 inches (12.7mm) minimum
- **Business card:** 0.75 inches (19mm) minimum

### Incorrect Logo Usage

**DO NOT:**

❌ Change logo colors outside approved palette
```
◇ BRANDORA  (in red, green, or other colors)
```

❌ Rotate or skew the logo
```
  /◇ BRANDORA/  (rotated)
```

❌ Add effects (shadows, gradients, outlines)
```
◇̲ ̲B̲R̲A̲N̲D̲O̲R̲A̲  (with drop shadow or outline)
```

❌ Stretch or distort proportions
```
◇ B R A N D O R A  (stretched horizontally)
```

❌ Place on busy backgrounds without proper contrast
```
[busy pattern] ◇ BRANDORA  (low visibility)
```

❌ Use low-resolution or pixelated versions

❌ Recreate or redraw the logo

❌ Place too close to other elements (violating clear space)

❌ Use unapproved logo variations or configurations

### Color Combinations

### Color Usage Guidelines

**Do:**
✓ Use Navy as primary brand color
✓ Use Blue for interactive elements
✓ Use Yellow sparingly for emphasis
✓ Maintain WCAG AA minimum (AAA preferred)
✓ Test color combinations for accessibility
✓ Use semantic colors consistently
✓ Provide sufficient contrast ratios

**Don't:**
✗ Use colors inconsistently across platforms
✗ Rely solely on color to convey information
✗ Use low-contrast combinations
✗ Use yellow for text or small elements
✗ Overuse accent colors
✗ Create new brand colors
✗ Use colors outside the defined palette

### Accessibility Compliance (WCAG 2.1)

All color combinations in this guide meet WCAG 2.1 Level AA standards minimum.

**AAA Compliant Combinations:**
- Navy (#070058) + White: 19.76:1
- White + Navy (#070058): 19.76:1
- Neutral 900 (#0F172A) + White: 15.52:1
- Yellow (#FFBE44) + Navy (#070058): 11.8:1
- Neutral 700 (#334155) + White: 10.87:1

**AA Compliant Combinations:**
- Blue (#6F8AEC) + White: 4.12:1 (Large text only)
- Neutral 500 (#64748B) + White: 4.73:1

**Testing Tools:**
- WebAIM Contrast Checker
- Stark plugin for Figma
- Chrome DevTools Color Picker
- Color Oracle (color blindness simulator)

---

### Color in Different Media

#### Digital/Screen
Use RGB and HEX values.
Colors will appear slightly more vibrant than print.

#### Print
Use CMYK values or Pantone spot colors.
Always request proof prints for color-critical materials.

#### Environmental (Signage, Merchandise)
Consult with vendor for color matching.
Provide Pantone references and digital color codes.

#### Video/Motion
Use RGB values.
Account for color shift in different video formats (export tests recommended).
