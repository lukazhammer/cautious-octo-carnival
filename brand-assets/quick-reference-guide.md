## Logo Usage

### Files
- **Primary:** `brandora-logo-primary.svg`
- **Reverse:** `brandora-logo-reverse.svg` (for dark backgrounds)
- **Icon:** `brandora-icon.svg`

### Minimum Sizes
- **Digital:** 150px width
- **Print:** 1.5 inches (38mm) width

### Clear Space
- Maintain space equal to height of letter "A" on all sides

### Don'ts
- ❌ Don't change colors
- ❌ Don't rotate or distort
- ❌ Don't add effects (shadows, gradients)
- ❌ Don't place on busy backgrounds without contrast

## Brand Voice

### Characteristics
- ✓ Clear (not clever)
- ✓ Empowering (not prescriptive)
- ✓ Confident (not arrogant)
- ✓ Conversational but professional
- ✓ Data-informed and human

### Tone by Context

| Context | Tone |
|---------|------|
| **Marketing** | Inspiring + Clear |
| **Product UI** | Helpful + Concise |
| **Education** | Insightful + Accessible |
| **Support** | Empathetic + Solution-Focused |
| **Social Media** | Engaging + Authentic |

## Icon Specifications

### Style
- **Type:** Outline/stroke
- **Weight:** 2px stroke (at 24×24)
- **Corners:** Rounded (2px radius)
- **Format:** SVG

### Sizes
- Small: 16×16px
- Medium: 24×24px (standard)
- Large: 32×32px
- XL: 48×48px+

### Colors
- Default: Neutral 700 `#334155`
- Hover/Interactive: Blue `#6F8AEC`
- Active: Navy `#070058`
- Disabled: Neutral 500 `#64748B` at 50% opacity

- **Background:** Blue `#6F8AEC`

## Spacing

**Use 8px increment system:**
- 4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px, 96px

**Common spacings:**
- Between paragraphs: 20-24px
- Between sections: 48-64px
- Card padding: 24-32px
- Button padding: 12-16px vertical, 24-32px horizontal

## Accessibility

### Color Contrast (WCAG 2.1)
- **AAA:** Navy on White (19.76:1) ✓
- **AA:** Blue on White (4.12:1) — Large text only
- **Body text:** Neutral 900 on White (15.52:1) ✓

### Best Practices
- Minimum 16px for body text
- Line height 1.5+ for body text
- Don't rely on color alone
- Provide alt text for images
- Keyboard navigable
- Focus indicators visible
