# Website Component Library

## Overview

**40 reusable components** organized by type:
- Headers/Navigation (5 variations)
- Hero Sections (8 variations)
- Feature Sections (6 variations)
- Pricing Tables (4 variations)
- Testimonial Cards (5 variations)
- CTA Sections (6 variations)
- Footers (3 variations)
- Forms (3 variations)

**Design principles:**
- Mobile-first responsive design
- Brand colors and typography
- Accessibility (WCAG AA minimum)
- Semantic HTML structure
- Reusable and modular

## Design System Integration

### Spacing Scale
- **4px:** 0.25rem
- **8px:** 0.5rem
- **12px:** 0.75rem
- **16px:** 1rem
- **24px:** 1.5rem
- **32px:** 2rem
- **48px:** 3rem
- **64px:** 4rem
- **96px:** 6rem

### Breakpoints
- **Mobile:** < 768px
- **Tablet:** 768px - 1023px
- **Desktop:** 1024px - 1439px
- **Large Desktop:** ≥ 1440px

### Container Max Width
- **Standard:** 1200px
- **Wide:** 1400px
- **Narrow:** 800px

### Component States
- **Default:** Standard appearance
- **Hover:** Subtle color shift, slight scale or shadow
- **Active:** Pressed state, darker color
- **Focus:** Blue outline for accessibility
- **Disabled:** Reduced opacity (50%), cursor not-allowed

## Implementation Notes

### CSS Framework
Consider using:
- Tailwind CSS (utility-first)
- Custom CSS with variables
- Styled-components (React)

### Accessibility
- Semantic HTML (header, nav, main, footer)
- ARIA labels where needed
- Keyboard navigation support
- Focus indicators
- Alt text for images
- Color contrast compliance

### Performance
- Lazy load images
- Optimize image sizes
- Minimize CSS/JS
- Critical CSS inline
- Defer non-critical scripts

---

This component library provides a comprehensive foundation for building consistent, accessible, and professional Brandora web experiences while maintaining flexibility for different use cases and pages.
