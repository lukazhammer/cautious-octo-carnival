## Design File Specifications

### File Organization
```
/social-media/
  /instagram/
    /posts/
      quote-minimal.psd
      quote-bold.psd
      tip-card-numbered.psd
      ...
    /stories/
      poll-template.psd
      question-box.psd
      ...
  /linkedin/
    thought-leadership-quote.psd
    data-statistic.psd
    ...
  /twitter/
    thread-header.psd
    quote-card.psd
    ...
  /exports/
    (Final JPG/PNG files)
```

### Layering Best Practices
- Keep layers organized and named
- Use layer groups for sections
- Include text as editable type layers
- Smart objects for images
- Layer comps for variations

### Export Settings
- **JPG:** Quality 90%, sRGB, optimized
- **PNG:** PNG-24, transparency if needed
- **File naming:** `platform_template-name_variation.jpg`

This template library ensures consistent, professional social media presence while allowing flexibility for different content types and platforms.
