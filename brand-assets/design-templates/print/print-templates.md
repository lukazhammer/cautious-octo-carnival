---

## Print Specifications

### Color Mode
- **CMYK** for print (not RGB)
- Pantone spot colors for brand consistency (optional, budget allowing)
- Include color profiles

### Resolution
- **300 DPI** minimum for print
- **600 DPI** for fine details (business cards, small text)

### Bleed
- **3mm (0.125")** bleed on all sides
- Extend background colors/images beyond trim line
- Keep important content 5mm inside trim line (safe area)

### File Formats
- **Design files:** AI, INDD, PDF (high-res)
- **Print-ready:** PDF/X-1a or PDF/X-4
- **Proofing:** PDF with crop marks

- Suggested margins: 25mm top, 20mm sides, 20mm bottom

## Print Production Checklist

Before sending to print:
- [ ] All text converted to outlines (or fonts embedded)
- [ ] CMYK color mode (not RGB)
- [ ] 300 DPI resolution (images and final file)
- [ ] Bleed added (3mm / 0.125")
- [ ] Safe area respected (5mm margin)
- [ ] No transparency issues
- [ ] PDF/X-1a or PDF/X-4 format
- [ ] Crop marks and bleeds visible
- [ ] Color profile embedded
- [ ] Proofread for typos
- [ ] Request printed proof before bulk printing
- [ ] Verify paper stock and finish
