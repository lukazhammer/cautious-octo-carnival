## Design System for Emails

### Layout Structure
```
┌────────────────────────────────┐
│         Header (Logo)          │
├────────────────────────────────┤
│        Hero/Banner Image       │
├────────────────────────────────┤
│                                │
│         Main Content           │
│                                │
├────────────────────────────────┤
│       Call-to-Action           │
├────────────────────────────────┤
│           Footer               │
└────────────────────────────────┘
```

### Typography
- **Headlines:** Arial/Helvetica Bold, 28-32px (Montserrat not always reliable in email)
- **Subheads:** Arial/Helvetica Bold, 20-24px
- **Body:** Arial/Helvetica, 16-18px, line-height 1.6
- **CTA buttons:** Arial/Helvetica Bold, 16-18px
- **Footer:** Arial/Helvetica, 14px

### Colors
- **Header bg:** White #FFFFFF or Background Blue #F1F9FF
- **Body bg:** White #FFFFFF or Background Blue #F1F9FF
- **Text:** Navy #070058 or Neutral 900 #0F172A
- **Links:** Blue #6F8AEC
- **CTA buttons:** Blue #6F8AEC or Navy #070058
- **Button text:** White #FFFFFF

### Spacing
- **Container padding:** 20-30px
- **Section spacing:** 30-40px
- **Paragraph spacing:** 16-20px
- **Button padding:** 14px 32px

## Email Design Best Practices

### Mobile Optimization
- Single column layout (600px max width)
- Large tap targets (buttons minimum 44×44px)
- Readable font sizes (minimum 14px body text)
- Test on multiple devices

### Accessibility
- Descriptive alt text for all images
- Sufficient color contrast (WCAG AA minimum)
- Semantic HTML structure
- Plain text version available
- Avoid relying solely on color

### Deliverability
- Keep image-to-text ratio reasonable (not image-only)
- Avoid spam trigger words in subject lines
- Include plain text version
- Clear unsubscribe link
- Authenticated sending domain

### Subject Lines
- 40-50 characters ideal
- Front-load value
- Personalization when relevant
- A/B test variations
- Avoid all caps, excessive punctuation

### Preview Text
- First 85-100 characters
- Complement subject line
- Add context or value
- Not same as subject line

## Email Footer Standard

**Required elements:**
- Company name and logo (small)
- Physical mailing address (CAN-SPAM compliance)
- Unsubscribe link (clear, functional)
- Update preferences link
- Social media icons/links
- Contact email

**Optional elements:**
- Quick links (Help, Blog, Pricing)
- Legal links (Privacy, Terms)
- "View in browser" link

**Design:**
- Small text (12-14px)
- Neutral color (Neutral 500 #64748B)
- Centered or left-aligned
- Minimal but complete

## Testing Checklist

Before sending:
- [ ] Subject line under 50 characters
- [ ] Preview text optimized
- [ ] All links working
- [ ] Images loading (with alt text)
- [ ] CTA buttons prominent and working
- [ ] Renders correctly on mobile
- [ ] Renders in major email clients (Gmail, Outlook, Apple Mail)
- [ ] Personalization tokens working
- [ ] Unsubscribe link functional
- [ ] Plain text version reviewed
- [ ] Spam score checked
- [ ] Sender name and email correct
- [ ] Proofread for typos
