# Values & Purpose Prompts

## Core Values Development (25 prompts)

### Values Identification

1. **Values Discovery**
   ```
   Help me identify core values that align with:
   - Our vision: "[vision statement]"
   - Our target audience: "[audience description]"
   - Our industry: "[industry]"
   - Our unique approach: "[what makes us different]"

   Suggest 10-15 potential core values with brief definitions.
   ```

2. **Values from Culture**
   ```
   Based on our existing culture, identify our implicit values:

   What we celebrate: [examples of recognized behaviors]
   What we reject: [behaviors that are discouraged]
   How we work: [work style and practices]
   What we prioritize: [time, resources, focus areas]

   Extract 5-7 core values reflected in these behaviors.
   ```

3. **Industry-Specific Values**
   ```
   What core values are particularly important for [industry] companies serving [audience]?

   Consider:
   - Industry best practices
   - Customer expectations
   - Regulatory requirements
   - Competitive landscape
   - Stakeholder priorities

   Rank by importance and explain why each matters.
   ```

4. **Differentiated Values**
   ```
   Most companies in [industry] claim these values: [common values].

   Help me identify values that:
   - Actually differentiate us
   - Reflect our unique approach
   - Matter to our specific audience
   - Guide distinctive behaviors
   - Can't be claimed by everyone

   Suggest 5-7 truly differentiated values.
   ```

5. **Values from Founders**
   ```
   Our founder's personal values and beliefs:
   - [value/belief 1]
   - [value/belief 2]
   - [value/belief 3]

   How should these translate into organizational values that:
   - Scale beyond the founder
   - Guide employee behavior
   - Shape company culture
   - Inform business decisions
   ```

### Values Definition & Articulation

6. **Values Definitions**
   ```
   Generate clear, specific definitions for these values in our brand context:

   Values: [list of values]
   Our industry: [industry]
   Our approach: [unique characteristics]

   For each value, provide:
   - A one-sentence definition
   - What it means in practice
   - What it doesn't mean (boundaries)
   - Example behaviors that demonstrate it
   ```

7. **Values Hierarchy**
   ```
   I have these potential values: [list]

   Help me:
   - Identify which are truly CORE (non-negotiable)
   - Which are aspirational (want to embody)
   - Which are operational (how we work)
   - Recommend a hierarchy or framework
   - Reduce to 3-5 core values

   Explain the reasoning for prioritization.
   ```

8. **Values Stories**
   ```
   For this core value: "[value name and definition]"

   Create:
   - A story that illustrates this value in action
   - A scenario where this value guided a tough decision
   - An example of what it looks like in customer interactions
   - A case of what violating this value would mean

   Make it specific to our [industry/context].
   ```

9. **Values Translation**
   ```
   Translate this abstract value into concrete behaviors:

   Value: "[value name]"
   Definition: "[definition]"

   What does this mean for:
   - How we hire (qualities we seek)
   - How we serve customers (service standards)
   - How we make decisions (decision criteria)
   - How we measure success (what we track)
   - How we resolve conflicts (conflict resolution)
   ```

10. **Values Language**
    ```
    I have these core values but they sound generic: [values list]

    Help me:
    - Use more distinctive language
    - Make them memorable and ownable
    - Avoid corporate clich√©s
    - Reflect our actual personality
    - Create unique names if needed

    Give 3 variations for each value.
    ```

### Values Validation & Testing

11. **Values Conflict Analysis**
    ```
    Identify potential conflicts or tensions between these values: [values list]

    For each conflict:
    - Describe the tension
    - Give examples of when they might clash
    - Suggest how to navigate the conflict
    - Determine if both values can coexist
    - Recommend priority or balance
    ```

12. **Values-Strategy Alignment**
    ```
    Do these values align with our strategy?

    Values: [list]
    Strategy: [strategic priorities and goals]
    Target audience: [description]

    Evaluate:
    - Which values support our strategy?
    - Which might create friction?
    - Are any values missing?
    - Should strategy or values shift?
    ```

13. **Values Authenticity Test**
    ```
    Test these values for authenticity: [values list]

    Questions:
    - Do we actually live these today?
    - Would employees recognize these as real?
    - Do our actions support these claims?
    - What evidence exists for each value?

    Be brutally honest about gaps between stated and lived values.
    ```

14. **Values Future-Proofing**
    ```
    Will these values remain relevant as we grow?

    Current state: [size, stage, market]
    Future state: [growth plans, expansion, evolution]
    Values: [list]

    Assess:
    - Which values will scale?
    - Which might need to evolve?
    - Which are fundamental regardless of size?
    - Should any be aspirational?
    ```

15. **Stakeholder Values Mapping**
    ```
    Map how these values resonate with different stakeholders:

    Values: [list]
    Stakeholders:
    - Customers: [description]
    - Employees: [description]
    - Partners: [description]
    - Investors: [description]

    Which values matter most to each group? Any conflicts?
    ```

## Brand Purpose Development (25 prompts)

### Purpose Discovery

16. **Purpose from Problem**
    ```
    Create a brand purpose statement connecting:

    The problem: [problem we solve]
    The solution: [how we solve it]
    The impact: [change we create]
    Who benefits: [stakeholders]

    Format: "We exist to [purpose] so that [impact]."
    ```

17. **Purpose Beyond Profit**
    ```
    Our business makes money by: [business model]

    But our deeper purpose is:
    - What change we want to see in the world
    - How we improve lives beyond our product
    - What would be lost if we didn't exist
    - The legacy we want to leave

    Articulate a purpose that transcends profit.
    ```

18. **Customer Transformation Purpose**
    ```
    Define our purpose based on customer transformation:

    Customers come to us: [initial state]
    We help them become: [transformed state]
    This matters because: [larger significance]
    The world becomes: [systemic impact]

    Create a purpose statement focused on this transformation.
    ```

19. **Category Purpose**
    ```
    What is our purpose within the [category/industry] space?

    Current category state: [problems, gaps, status quo]
    Our unique role: [what we uniquely contribute]
    Category future: [where we're leading it]

    Purpose should clarify our role in shaping the category.
    ```

20. **Founder's Why**
    ```
    Why did we start this company?

    Original motivation: [founding story]
    Personal experience: [relevant background]
    Frustration with: [what existed before]
    Vision for: [better alternative]

    Distill this into a compelling purpose statement.
    ```

### Purpose Refinement

21. **Purpose-Vision Alignment**
    ```
    Validate alignment between purpose and vision:

    Purpose: "[purpose statement]"
    Vision: "[vision statement]"

    Do they:
    - Support each other logically?
    - Tell a coherent story together?
    - Address different aspects appropriately?
    - Create clear direction?

    Refine either or both for better alignment.
    ```

22. **Purpose Specificity**
    ```
    Make this purpose more specific and actionable: "[vague purpose]"

    Add:
    - Specific beneficiaries
    - Tangible outcomes
    - Clear change we create
    - Measurable impact

    While keeping it inspirational and enduring.
    ```

23. **Purpose Differentiation**
    ```
    My purpose sounds too similar to competitors: "[current purpose]"

    Competitors' purposes:
    - [Competitor A purpose]
    - [Competitor B purpose]

    Differentiate by:
    - Emphasizing unique aspects
    - Using distinctive framing
    - Connecting to specific approach
    - Highlighting unique impact
    ```

24. **Purpose Simplification**
    ```
    Simplify this complex purpose: "[long purpose statement]"

    Reduce to:
    - One powerful sentence
    - Under 15 words
    - No jargon
    - Clear and memorable
    - Emotionally resonant

    Maintain the core meaning.
    ```

25. **Purpose Expansion**
    ```
    Expand this narrow purpose to broader impact: "[limited purpose]"

    Consider:
    - Second and third-order effects
    - Ripple impacts beyond direct customers
    - Contribution to industry/society
    - Long-term legacy

    Without becoming too abstract or disconnected from core business.
    ```

### Purpose Application

26. **Purpose in Marketing**
    ```
    How should this purpose show up in marketing?

    Purpose: "[purpose statement]"
    Target audience: [description]

    Create:
    - A tagline that hints at purpose
    - Key messaging pillars
    - Content themes
    - Campaign concepts
    - Story angles
    ```

27. **Purpose in Product**
    ```
    How does this purpose guide product development?

    Purpose: "[purpose statement]"
    Current products: [list]

    Answer:
    - Which products best embody our purpose?
    - What's missing to fully deliver on purpose?
    - What features align with purpose?
    - What should we stop doing?
    - Future product direction?
    ```

28. **Purpose in Culture**
    ```
    Translate this purpose into cultural practices:

    Purpose: "[purpose statement]"

    How should it influence:
    - Hiring criteria and process
    - Onboarding and training
    - Performance evaluation
    - Recognition and rewards
    - Decision-making frameworks
    ```

29. **Purpose Metrics**
    ```
    What should we measure to track delivery on our purpose?

    Purpose: "[purpose statement]"

    Define:
    - Direct impact metrics
    - Proxy indicators
    - Qualitative measures
    - Stakeholder feedback
    - Progress milestones

    Balance inspiration with accountability.
    ```

30. **Purpose Communication**
    ```
    Create a communication framework for our purpose:

    Purpose: "[purpose statement]"

    Develop:
    - Elevator pitch version
    - Detailed explanation
    - Customer-facing version
    - Employee-facing version
    - Investor-facing version
    - FAQ addressing common questions
    ```

### Advanced Purpose Work

31. **Purpose Evolution**
    ```
    Should our purpose evolve?

    Original purpose: "[founding purpose]"
    Current reality: [how business has changed]
    Market changes: [relevant shifts]

    Assess:
    - Is the original purpose still relevant?
    - Should it expand or refine?
    - What should stay constant?
    - What might need to shift?
    ```

32. **Multi-Stakeholder Purpose**
    ```
    Create a purpose that serves multiple stakeholders:

    Customers need: [outcomes]
    Employees seek: [fulfillment]
    Community needs: [contribution]
    Planet needs: [sustainability]

    Craft a purpose that integrates these without being generic.
    ```

33. **Purpose in Adversity**
    ```
    Test if our purpose holds during challenges:

    Purpose: "[purpose statement]"
    Challenge scenarios:
    - Economic downturn
    - Competitive pressure
    - Resource constraints
    - Market shifts

    Does purpose:
    - Guide decisions in tough times?
    - Provide resilience and focus?
    - Remain relevant under pressure?
    ```

34. **Purpose Validation**
    ```
    Validate this purpose with key stakeholders:

    Purpose: "[purpose statement]"

    Questions for:
    - Customers: Does this resonate with why they choose us?
    - Employees: Does this inspire meaningful work?
    - Leadership: Does this guide strategy?
    - Partners: Does this align with collaboration?

    Gather and synthesize feedback.
    ```

35. **Purpose Legacy**
    ```
    What legacy would this purpose create?

    Purpose: "[purpose statement]"

    If we fully delivered on this purpose over 20 years:
    - How would the world be different?
    - What would be our lasting contribution?
    - How would we be remembered?
    - What impact would endure?

    Is this the legacy we want? Adjust if needed.
    ```

## Integration: Values + Purpose

### Alignment Prompts

36. **Values-Purpose Connection**
    ```
    How do our values support our purpose?

    Purpose: "[purpose statement]"
    Values: [list]

    For each value, explain:
    - How it enables our purpose
    - What would happen without this value
    - Specific ways it supports purpose delivery
    ```

37. **Values-Purpose Conflicts**
    ```
    Identify tensions between values and purpose:

    Purpose: "[purpose statement]"
    Values: [list]

    Where might:
    - A value conflict with purpose?
    - Short-term value clash with long-term purpose?
    - Internal value conflict with external purpose?

    How to resolve these tensions?
    ```

38. **Unified Brand Foundation**
    ```
    Create a cohesive brand foundation narrative:

    Vision: "[vision statement]"
    Mission: "[mission statement]"
    Purpose: "[purpose statement]"
    Values: [list]

    Write a 2-3 paragraph narrative showing how these connect and tell one coherent brand story.
    ```

39. **Decision Framework**
    ```
    Create a decision-making framework using our brand foundation:

    Vision: "[vision]"
    Mission: "[mission]"
    Purpose: "[purpose]"
    Values: [list]

    For major decisions, evaluate:
    - Does it advance our vision?
    - Does it fit our mission?
    - Does it serve our purpose?
    - Does it honor our values?

    Provide a scoring or evaluation method.
    ```

40. **Complete Brand Platform**
    ```
    Review our complete brand platform for coherence:

    Vision: "[vision]"
    Mission: "[mission]"
    Purpose: "[purpose]"
    Values: [list]
    Positioning: "[positioning]"

    Assess:
    - Internal consistency
    - Logical flow
    - Mutual reinforcement
    - Gaps or redundancies
    - Overall strength

    Suggest refinements for a powerful, unified platform.
    ```

## How to Use These Prompts

### For Values Work

1. **Discovery Phase**: Use prompts 1-5 to identify potential values
2. **Definition Phase**: Use prompts 6-10 to articulate and refine
3. **Validation Phase**: Use prompts 11-15 to test and confirm

### For Purpose Work

1. **Exploration**: Use prompts 16-20 to discover purpose
2. **Refinement**: Use prompts 21-25 to clarify and strengthen
3. **Application**: Use prompts 26-30 to operationalize
4. **Evolution**: Use prompts 31-35 for ongoing development

### For Integration

1. Use prompts 36-40 to ensure alignment across your entire brand foundation
2. Regular review (annually) to maintain coherence
3. Test during major decisions or strategic shifts

## Best Practices

1. **Start Broad, Then Narrow**: Generate many options before selecting few
2. **Be Honest**: Your values should reflect reality, not aspiration (unless explicitly labeled aspirational)
3. **Test Rigorously**: Values and purpose must withstand scrutiny and challenging scenarios
4. **Involve Stakeholders**: Get input from leadership, employees, and customers
5. **Document Examples**: Collect stories and examples that bring values and purpose to life
6. **Review Regularly**: Revisit annually to ensure continued relevance
7. **Live It**: Values and purpose must be demonstrated, not just stated
