# Vision & Mission Statement Prompts

## Vision Statement Development (25 prompts)

### Analysis & Refinement

1. **Vision Analysis**
   ```
   Analyze this vision statement and suggest improvements for clarity, inspiration, and strategic direction: "[your vision statement]"

   Evaluate it against these criteria:
   - Is it aspirational and inspiring?
   - Does it paint a clear picture of the future?
   - Is it memorable and concise?
   - Does it differentiate us from competitors?
   ```

2. **Vision Gap Analysis**
   ```
   I have this vision: "[your vision]" but my current positioning is: "[current state]".

   Identify:
   - The gaps between vision and current reality
   - What's missing to bridge this gap
   - Specific steps to align positioning with vision
   - Potential conflicts or contradictions
   ```

3. **Vision Validation**
   ```
   Test this vision statement for effectiveness: "[vision statement]"

   Does it:
   - Inspire employees and stakeholders?
   - Provide clear strategic direction?
   - Align with our target audience's values?
   - Support long-term business goals?

   Suggest improvements if needed.
   ```

4. **Vision Simplification**
   ```
   My vision statement is too complex: "[long vision]"

   Help me:
   - Distill it to its essence (under 20 words)
   - Remove jargon and corporate speak
   - Make it memorable and quotable
   - Maintain the core meaning
   ```

5. **Vision Expansion**
   ```
   My vision is too vague: "[vague vision]"

   Help me expand it by:
   - Adding specific outcomes we're working toward
   - Clarifying who we serve and how
   - Describing the future state we're creating
   - Making the impact more tangible
   ```

### Vision Generation

6. **Industry-Specific Vision**
   ```
   Generate 5 compelling vision statement variations for a [industry] company that:
   - Serves [target audience]
   - Solves [primary problem]
   - Operates in [geographic region/market]
   - Values [core values]

   Each should be inspiring, clear, and differentiated.
   ```

7. **Purpose-Driven Vision**
   ```
   Create a vision statement for a company that exists to [primary purpose/reason for being].

   The vision should:
   - Connect to a larger societal benefit
   - Inspire employees to meaningful work
   - Attract customers who share our values
   - Guide strategic decision-making
   ```

8. **Customer-Centric Vision**
   ```
   Craft a vision statement focused on the transformation we create for [customer type].

   Currently they experience: [pain points]
   We help them achieve: [desired outcomes]
   The future we're building: [ideal state]

   Make it customer-focused, not company-focused.
   ```

9. **Innovation Vision**
   ```
   Generate a vision statement for an innovative company disrupting [industry] through [innovation/approach].

   Emphasize:
   - The change we're bringing to the industry
   - The better future we're enabling
   - Why this matters beyond profit
   - Our unique approach or perspective
   ```

10. **Market Position Vision**
    ```
    Create a vision statement that positions us as [desired market position] in [industry/category].

    We want to be known for: [key differentiators]
    We serve: [specific niche]
    We're different because: [unique approach]

    Balance ambition with credibility.
    ```

### Mission Statement Development (25 prompts)

### Mission Generation

11. **Core Mission Statement**
    ```
    Generate 5 mission statement variations for a [industry] company serving [target audience] that:

    - Provides: [products/services]
    - Solves: [problems]
    - Delivers value through: [unique approach]
    - Operates with: [key values/principles]

    Each should be specific, actionable, and authentic.
    ```

12. **Concise Mission**
    ```
    Create a one-sentence mission statement that captures:
    - What we do: [core offering]
    - Who we serve: [target audience]
    - How we're different: [unique approach]
    - Why it matters: [value delivered]

    Keep it under 25 words and make every word count.
    ```

13. **Detailed Mission**
    ```
    Develop a comprehensive mission statement (2-3 sentences) that clearly articulates:

    1. Our primary business activity
    2. Our target customers/beneficiaries
    3. Our unique value proposition
    4. Our core operating principles
    5. The impact we create
    ```

14. **Mission from Vision**
    ```
    Given this vision: "[vision statement]"

    Create a mission statement that:
    - Describes what we DO to achieve that vision
    - Specifies our target market and offerings
    - Explains our unique approach
    - Provides clear direction for operations
    ```

15. **Service-Based Mission**
    ```
    Craft a mission statement for a service business that:

    Services offered: [list]
    Client profile: [description]
    Methodology: [approach]
    Outcomes delivered: [results]

    Make it specific enough to guide operations but flexible enough to evolve.
    ```

### Mission Refinement

16. **Mission Specificity**
    ```
    Refine this mission to be more specific and actionable: "[current mission]"

    Make it:
    - More concrete (less abstract)
    - More measurable (quantifiable where possible)
    - More differentiated (clearly unique)
    - More directional (guides decisions)
    ```

17. **Mission Authenticity Check**
    ```
    Evaluate this mission statement for authenticity: "[mission]"

    Questions:
    - Does it sound like generic corporate speak?
    - Could any competitor claim the same mission?
    - Does it reflect our true operations?
    - Would employees recognize this as genuine?

    Suggest ways to make it more authentic.
    ```

18. **Mission-Operations Alignment**
    ```
    I have this mission: "[mission statement]" but my actual operations are: "[what you actually do]"

    Either:
    A) Revise the mission to match reality, or
    B) Identify gaps and suggest operational changes

    Which approach makes more sense and why?
    ```

19. **Mission Modernization**
    ```
    This mission statement feels outdated: "[old mission]"

    Update it to:
    - Reflect current market realities
    - Use contemporary language (not corporate jargon)
    - Address modern customer needs
    - Incorporate new capabilities or offerings

    Keep the core essence but make it relevant for today.
    ```

20. **Mission Differentiation**
    ```
    My mission sounds too similar to competitors: "[current mission]"

    Competitor missions:
    - Competitor A: "[their mission]"
    - Competitor B: "[their mission]"

    Help me differentiate by:
    - Identifying our unique approach
    - Emphasizing different aspects
    - Using distinctive language
    - Highlighting what only we do
    ```

### Mission Testing & Validation

21. **Stakeholder Perspective**
    ```
    Evaluate this mission from multiple stakeholder perspectives: "[mission]"

    How does it resonate with:
    - Customers: Does it promise value they care about?
    - Employees: Does it inspire meaningful work?
    - Partners: Does it suggest good collaboration?
    - Investors: Does it indicate sustainable business?

    Adjust based on stakeholder priorities.
    ```

22. **Decision-Making Filter**
    ```
    Test if this mission statement provides clear decision-making criteria: "[mission]"

    Use it to evaluate these decisions:
    - Should we launch [new product/service]?
    - Should we enter [new market]?
    - Should we partner with [company type]?

    Does the mission help answer these questions clearly?
    ```

23. **Mission Longevity**
    ```
    Assess whether this mission will remain relevant: "[mission]"

    Consider:
    - Will it still be true in 5 years? 10 years?
    - Is it tied to temporary trends?
    - Does it allow for business evolution?
    - Is it fundamental enough to endure?

    Suggest adjustments for longevity.
    ```

24. **Cultural Alignment**
    ```
    Evaluate alignment between this mission and our culture: "[mission]"

    Our culture is characterized by:
    - [cultural trait 1]
    - [cultural trait 2]
    - [cultural trait 3]

    Does the mission:
    - Reflect our actual culture?
    - Reinforce desired behaviors?
    - Guide cultural evolution?
    ```

25. **Mission Implementation**
    ```
    Given this mission: "[mission statement]"

    How should it influence:
    - Hiring criteria (who we hire)
    - Product development (what we build)
    - Customer service (how we serve)
    - Marketing messaging (what we say)
    - Strategic planning (where we go)

    Provide specific examples for each area.
    ```

## How to Use These Prompts

### Best Practices

1. **Be Specific**: Replace bracketed placeholders with your actual information
2. **Iterate**: Use multiple prompts to refine your statements
3. **Combine**: Use analysis prompts after generation prompts
4. **Test**: Validate with stakeholders before finalizing
5. **Context**: Provide background information for better results

### Workflow Suggestion

1. Start with generation prompts (6-10, 11-15)
2. Refine with improvement prompts (16-20)
3. Validate with testing prompts (21-25)
4. Iterate based on feedback
5. Finalize and document

### AI Tool Recommendations

- **Perplexity**: Use for market research and competitive analysis before crafting statements
- **Claude/ChatGPT**: Use for generation, refinement, and iteration
- **Multiple tools**: Get different perspectives by trying the same prompt across different AI tools

## Tips for Better Results

1. **Provide Context**: The more background you share, the better the output
2. **Be Honest**: Share your actual situation, not idealized version
3. **Ask for Variations**: Request multiple options to compare
4. **Iterate**: Refine through multiple conversations
5. **Human Review**: Always have leadership review and approve
