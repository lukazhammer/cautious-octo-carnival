# Content Strategy Prompts

## Content Planning & Strategy (25 prompts)

### Content Ideation

1. **Blog Post Ideas Generator**
   ```
   Generate 20 blog post ideas for a [industry] brand:

   Target audience: [description]
   Brand expertise: [topics you know well]
   Business goals: [awareness, leads, sales, education]
   SEO keywords: [target keywords]
   Competitor content gaps: [what they're not covering]

   For each idea, provide:
   - Compelling headline
   - Target keyword
   - Content angle/hook
   - Estimated search volume/interest
   - Difficulty level (beginner, intermediate, advanced)

   Mix of:
   - How-to guides
   - Thought leadership
   - Case studies/examples
   - Industry trends
   - Problem-solution posts
   ```

2. **Content Calendar Creation**
   ```
   Create a content calendar for [timeframe] targeting [audience]:

   Goals: [what you want to achieve]
   Topics: [main themes to cover]
   Channels: [blog, social, email, etc.]
   Frequency: [how often to publish]

   For each piece, specify:
   - Publication date
   - Content title/topic
   - Format (blog, video, infographic, etc.)
   - Target audience segment
   - Primary CTA and goal
   - Promotion channels

   Consider:
   - Seasonal relevance
   - Product launches or events
   - Content variety and balance
   - Resource requirements
   ```

3. **Content Pillar Strategy**
   ```
   Develop a pillar content strategy for [topic area]:

   Main pillar topic: [broad topic]
   Target audience: [who you're serving]
   Business objective: [what you want to achieve]

   Create structure:
   - 1 comprehensive pillar page (ultimate guide)
   - 8-12 cluster content pieces (supporting topics)
   - Internal linking strategy
   - Conversion paths

   For pillar page:
   - Outline major sections
   - Key questions to answer
   - Unique value/angle
   - SEO opportunities

   For each cluster:
   - Specific topic
   - How it relates to pillar
   - Target keyword
   - Content format
   ```

4. **Content Themes Aligned with Values**
   ```
   Suggest content themes aligned with our brand values:

   Brand values: [list values]
   Target audience: [description]
   Industry: [your industry]
   Content goal: [awareness, education, conversion]

   For each value, develop:
   - 3-5 content themes
   - How theme expresses the value
   - Example topics within each theme
   - Best formats for this theme
   - Frequency/cadence

   Ensure themes:
   - Resonate with audience
   - Showcase expertise
   - Drive business goals
   - Can sustain ongoing content
   ```

5. **Content Audit and Gap Analysis**
   ```
   Audit our existing content and identify gaps:

   Current content: [what you have]
   Target keywords: [SEO targets]
   Customer journey stages: [awareness, consideration, decision]
   Competitor content: [what they cover]

   Analyze:
   - What's performing well (keep, update)
   - What's underperforming (improve or remove)
   - What's missing (gaps to fill)
   - What's outdated (refresh priority)

   Gap analysis:
   - Topics competitors cover that we don't
   - Journey stages lacking content
   - Keywords we should target
   - Formats we're underusing

   Prioritized content creation plan.
   ```

### Content Scoring Matrix

```
Score content ideas on:
- Audience Interest (1-5)
- SEO Opportunity (1-5)
- Differentiation (1-5)
- Resource Requirement (1-5, lower is better)
- Business Impact (1-5)

Total Score = Sum / Number of criteria
Prioritize highest scores
```

## Best Practices

1. **Audience-First**: Create for your audience, not for you
2. **Quality > Quantity**: Better to publish less but better
3. **Consistent Voice**: Maintain brand personality across all content
4. **SEO Foundation**: Optimize but write for humans
5. **Repurpose**: Get maximum value from each piece
6. **Promote**: Creation is 20%, promotion is 80%
7. **Measure**: Track what works and do more of it
8. **Update**: Refresh high-performing content regularly
9. **Document**: Use templates, guides, and workflows
10. **Iterate**: Continuously improve based on data

## Content Types Glossary

- **Pillar Content**: Comprehensive, authoritative guide on broad topic
- **Cluster Content**: Supporting pieces that dive deep into subtopics
- **Evergreen**: Content that remains relevant over time
- **Timely/Newsjacking**: Content tied to current events
- **Thought Leadership**: Opinion and insights from leadership
- **How-To**: Step-by-step instructional content
- **Listicle**: Numbered or bulleted list format
- **Case Study**: Customer success story with results
- **Ultimate Guide**: Exhaustive resource on a topic
- **Content Upgrade**: Bonus content offered in exchange for email
