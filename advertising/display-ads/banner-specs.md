# Display Advertising Banner Specifications

## Banner Sizes & Standard Specs

### 1. Leaderboard (728×90)
**Placement:** Top of web pages, above content
**File Specs:**
- Dimensions: 728 x 90 pixels
- File Format: JPG, PNG, GIF, HTML5
- Max File Size: 150KB
- Animation: 30 seconds max, 5 fps max
- Safe Zone: 10px margin all sides

### 2. Medium Rectangle (300×250)
**Placement:** Sidebar, inline content, mobile
**File Specs:**
- Dimensions: 300 x 250 pixels
- File Format: JPG, PNG, GIF, HTML5
- Max File Size: 150KB
- Animation: 30 seconds max, 5 fps max
- Safe Zone: 10px margin all sides

### 3. Wide Skyscraper (160×600)
**Placement:** Right sidebar, vertical spaces
**File Specs:**
- Dimensions: 160 x 600 pixels
- File Format: JPG, PNG, GIF, HTML5
- Max File Size: 150KB
- Animation: 30 seconds max, 5 fps max
- Safe Zone: 10px margin all sides

### 4. Half Page (300×600)
**Placement:** Sidebar, large vertical spaces
**File Specs:**
- Dimensions: 300 x 600 pixels
- File Format: JPG, PNG, GIF, HTML5
- Max File Size: 150KB
- Animation: 30 seconds max, 5 fps max
- Safe Zone: 10px margin all sides

### 5. Mobile Banner (320×50)
**Placement:** Mobile web bottom/top
**File Specs:**
- Dimensions: 320 x 50 pixels
- File Format: JPG, PNG, GIF, HTML5
- Max File Size: 150KB
- Animation: 30 seconds max, 5 fps max
- Safe Zone: 5px margin all sides

## Design Guidelines for All Banners

### Brand Colors:
- **Primary:** Navy #070058
- **Secondary:** Blue #6F8AEC
- **Accent:** Yellow #FFBE44
- **Background:** White #FFFFFF or Light Blue #F1F9FF
- **Text:** Navy #070058 or Near Black #0F172A

### Typography:
- **Headlines:** Montserrat Semi Bold 600
- **Body:** Poppins Regular 400
- **CTA:** Montserrat Semi Bold 600

### Font Sizes (Adjust per banner size):
**728×90:**
- Headline: 24-30px
- Body: 14-16px
- CTA: 14-16px

**300×250:**
- Headline: 20-24px
- Body: 12-14px
- CTA: 14px

**160×600:**
- Headline: 18-22px
- Body: 12-14px
- CTA: 14px

**300×600:**
- Headline: 24-28px
- Body: 14-16px
- CTA: 16px

**320×50:**
- Headline: 12-14px
- CTA: 12px
- Minimal body text

### CTA Button Design:
- **Shape:** Rounded corners (4-6px radius)
- **Color:** Yellow #FFBE44 (primary) or Blue #6F8AEC (secondary)
- **Text Color:** Navy #070058 or White #FFFFFF (depending on background)
- **Padding:** 10-15px horizontal, 8-12px vertical
- **Hover State:** Slightly darker shade
- **Border:** None or 2px solid for outlined style

### Visual Elements:
- **Logo:** Include Brandora logo (top left or bottom right)
- **Imagery:** Professional, on-brand, not stock-y
- **Icons:** Use Brandora icon library when applicable
- **White Space:** Don't cram – let design breathe
- **Hierarchy:** Clear visual hierarchy (headline → body → CTA)

### Animation Guidelines (if using):
- **Duration:** Max 30 seconds total
- **Loop:** Max 3 times then stop on final frame
- **Frame Rate:** Max 5 fps
- **Transitions:** Smooth, professional (fade, slide)
- **Purpose:** Draw attention, not distract
- **Final Frame:** Must show CTA clearly

### Mobile Optimization:
- **Larger Touch Targets:** CTA button minimum 44x44px
- **Readable Text:** Minimum 12px font size
- **Simplified Design:** Less elements, clearer hierarchy
- **Fast Loading:** Optimize image sizes
- **Clear CTA:** Prominent, obvious action

## HTML5 Banner Specifications (Advanced)

### Technology Stack:
- HTML5, CSS3, JavaScript
- No Flash
- Responsive design
- Cross-browser compatible

### File Structure:
```
banner-name/
  index.html
  style.css
  script.js
  images/
    logo.svg
    background.jpg
  fonts/ (if using web fonts)
```

### Code Requirements:
- **ClickTag:** Must include for tracking
- **Polite Loading:** Load after page content
- **Fallback:** Static image backup
- **File Size:** Total package under 150KB
- **Compression:** Minify all code

### Example ClickTag Implementation:
```html
<div id="banner" onclick="window.open(window.clickTag)">
  <!-- Banner content -->
</div>
```

### Testing Requirements:
- Chrome, Firefox, Safari, Edge
- Desktop and mobile
- Various screen sizes
- Click tracking functionality
- Load time under 3 seconds

## A/B Testing Framework

### Test Variables:
1. **Headline:** Benefit vs feature vs question
2. **Visual:** Screenshot vs illustration vs photo
3. **CTA Text:** "Start Free" vs "Get Started" vs "Try Now"
4. **CTA Color:** Yellow vs blue vs green
5. **Layout:** Text-heavy vs image-heavy vs balanced
6. **Offer:** Free trial vs free assessment vs download guide

### Success Metrics:
- **CTR (Click-Through Rate):** Primary metric
- **Viewability:** % of impressions actually seen
- **Engagement Rate:** Hover, interaction
- **Conversion Rate:** Post-click actions
- **Cost Per Click:** Efficiency metric
- **ROAS (Return on Ad Spend):** Ultimate measure

### Testing Protocol:
- Run minimum 10,000 impressions per variation
- Test one variable at a time
- Minimum 1-week test duration
- Statistical significance before winner
- Document learnings for future campaigns

## Ad Network Specifications

### Google Display Network:
- Accepts: JPG, PNG, GIF, HTML5
- Max file size: 150KB
- Animation: 30 seconds max
- Responsive display ads: Provide multiple assets

### Facebook Audience Network:
- Accepts: JPG, PNG
- Max file size: 30MB
- Text limit: 20% of image
- Minimum resolution: Listed dimensions

### Microsoft Advertising:
- Accepts: JPG, PNG, GIF, HTML5
- Max file size: 150KB
- Animation: 30 seconds max, 5 loops max

### Programmatic Platforms:
- IAB Standard sizes preferred
- HTML5 for rich media
- VAST/VPAID for video
- Secure (HTTPS) assets
