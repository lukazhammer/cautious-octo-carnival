# Data Governance Policy

## Principles

1. **Privacy First**: User privacy is paramount
2. **Transparency**: Clear about data collection
3. **Security**: Protect all analytics data
4. **Compliance**: Follow GDPR, CCPA
5. **Quality**: Maintain data accuracy

## Data Retention

- **Events**: 2 years
- **User data**: Indefinite (while active)
- **Logs**: 90 days
- **Reports**: 3 years

## Access Control

- **Admin**: Analytics team only
- **Analyst**: View-only dashboards
- **Team leads**: Dept-specific dashboards

## Compliance

### GDPR
- Data export available
- Right to deletion
- Consent management
- Anonymization

### CCPA
- Opt-out option
- Data disclosure
- No selling of data
