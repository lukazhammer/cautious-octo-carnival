## SLA & Standards

### Report Delivery SLA
- **Daily Reports:** By 8:00 AM daily
- **Weekly Reports:** By 9:00 AM Monday
- **Monthly Reports:** By 12:00 PM on 1st business day
- **Quarterly QBR:** Within 5 business days of quarter end
- **Ad-hoc Reports:** Within agreed timeline

### Quality Standards
- âœ… Data accuracy verified
- âœ… Calculations validated
- âœ… Trends explained
- âœ… Actionable insights included
- âœ… Clear visualization
- âœ… Consistent branding

### Late Report Protocol
1. Notify recipients via email/Slack
2. Provide ETA for delivery
3. Escalate to Analytics Lead if >2 hours late
4. Root cause analysis if pattern emerges

## Report Request Process

### New Report Requests
1. Submit request in #analytics-requests
2. Include: Purpose, audience, frequency, metrics needed
3. Analytics team reviews within 2 business days
4. Estimate: Effort and timeline
5. Prioritize against roadmap
6. Implement if approved

### Changes to Existing Reports
1. Submit change request in #analytics-requests
2. Explain rationale and impact
3. Review by report owner
4. Implement if approved
5. Notify all recipients of changes
